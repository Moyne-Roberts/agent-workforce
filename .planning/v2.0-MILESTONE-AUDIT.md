### Integration Gaps (2)

**INT-01: Holdout dataset ID path mismatch (BREAK)**
- **Affects:** ITER-05 (re-test on holdout split)
- **Issue:** `iterator.md` Phase 7 Step 7.2 references `dataset.per_agent.{agent_key}.holdout_dataset_id` — field path does not exist in `test-results.json` v3.0 template. Template stores `dataset.per_agent_datasets[].dataset_id` without per-split IDs.
- **Fix:** Add `train_dataset_id`, `test_dataset_id`, `holdout_dataset_id` to `per_agent_datasets[]` entries in template and update tester.md Phase 5.3.

**INT-02: Flag convention inconsistency (MINOR)**
- **Affects:** GUARD-03
- **Issue:** `harden.md` uses positional `[agent-key]` arg while deploy/test/iterate use `--agent` flag. SKILL.md documents `--agent` for all 4, contradicting harden.md.
- **Fix:** Update harden.md Step 3 to use `--agent` flag.
