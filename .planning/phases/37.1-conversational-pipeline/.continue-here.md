<decisions_made>

- **Conversation agent replaces hardcoded logic**: One Orq.ai agent (Sonnet) handles all user interaction — discussion, summaries, confirm/feedback. Uses `<pipeline_action>` tags for structured decisions. No hardcoded string matching.
- **AI extracts agents for graph**: Haiku call extracts agents as JSON from architect output. Stored in result.agents alongside raw output. Graph-mapper prefers JSON, falls back to regex.
- **3-column layout**: graph | vertical progress timeline (180px) | chat (480px). Timeline has clickable stages with modal output popup.
- **Keep Inngest for durability**: Single long-running Orq.ai call too fragile for 15min pipeline. Inngest step.run() for stage execution, streaming outside step.run().
- **Roadmap split**: Phase 38.1 (9 missing pipeline agents) separated from Phase 43 (upstream sync mechanism only).
</decisions_made>
