Action 1 -- `storeCredential`:
```typescript
export async function storeCredential(formData: {
  name: string;
  authType: string;
  values: Record<string, string>;
  projectIds?: string[];
}): Promise<{ success?: boolean; credentialId?: string; error?: string }>
```
- Zod schema: `z.object({ name: z.string().min(1, "Name is required").max(200), authType: z.string().min(1), values: z.record(z.string(), z.string()).check(ctx => { if (Object.keys(ctx.value).length === 0) ctx.issues.push({ message: "At least one value required" }); }), projectIds: z.array(z.string().uuid()).optional() })`
- Authenticate: `const supabase = await createClient(); const { data: { user } } = await supabase.auth.getUser();`
- If no user: `return { error: "Not authenticated" }`
- Encrypt: `const encrypted = encryptCredential(JSON.stringify(parsed.data.values));`
- Insert via admin: `createAdminClient().from("credentials").insert({ name: parsed.data.name, auth_type: parsed.data.authType, encrypted_values: encrypted, status: "not_tested", created_by: user.id }).select("id").single()`
- If projectIds: insert into credential_project_links via admin
- `revalidatePath("/settings")`
- Return `{ success: true, credentialId: data.id }`

Action 2 -- `replaceCredential`:
```typescript
export async function replaceCredential(
  credentialId: string,
  values: Record<string, string>
): Promise<{ success?: boolean; error?: string }>
```
- Authenticate user
- Verify ownership: SELECT credential via authenticated client (RLS enforced)
- If not found: `return { error: "Credential not found" }`
- Encrypt new values: `encryptCredential(JSON.stringify(values))`
- Update via admin: `{ encrypted_values: encrypted, status: "not_tested", failed_at: null }`
- `revalidatePath("/settings")`
- Return `{ success: true }`

Action 3 -- `deleteCredential`:

function formatRelativeTime(dateStr: string): string {
  const diffSec = Math.floor((Date.now() - new Date(dateStr).getTime()) / 1000);
  if (diffSec < 60) return "just now";
  if (diffSec < 3600) return `${Math.floor(diffSec / 60)}m ago`;

  return new Date(dateStr).toLocaleDateString();
}
