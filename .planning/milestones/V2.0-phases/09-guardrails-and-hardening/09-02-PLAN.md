   - Command format becomes: `/orq-agent:deploy [--agent agent-key]` where `--agent` is optional

   - If NO `--agent` flag: show an interactive picker (LOCKED decision from CONTEXT.md):

     ```
       2. agent-key-1

       4. orchestrator-key

     ```
